DESCRIPTION OF THE PRODUCT

    Shopping list is a app where user can make a list of products which they are going
    to buy. The app allows user to add and remove products fromthe list. User can also 
    merge lists together and save the list to dropbox. Thelist can be used as a command 
    line app with CLI and with GUI which is made with Swing.
